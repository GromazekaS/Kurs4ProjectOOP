# Парсер вакансий с HeadHunter (hh.ru)

Проект позволяет искать, фильтровать и сохранять вакансии с платформы hh.ru в удобном формате. Программа предоставляет консольный интерфейс для работы с вакансиями и сохраняет результаты в JSON-файл.

## Основные возможности

- 🔍 Поиск вакансий по ключевым словам
- 🧩 Фильтрация по описанию и требованиям
- 📊 Сортировка по уровню зарплаты
- 💾 Сохранение результатов в JSON
- 🏆 Топ-N вакансий по зарплате
- ✔️ Валидация данных о зарплате
- ⚖️ Поддержка сравнения вакансий

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/yourusername/hh-vacancy-parser.git
cd hh-vacancy-parser
```
Установите зависимости:

```bash
poetry install
```
Запустите программу:

```bash
python main.py
```
Пример использования
```bash
Введите поисковый запрос (например: Python): Data Scientist
Введите количество вакансий для вывода в топ N: 5
Введите ключевые слова для фильтрации вакансий (через пробел): ML Python SQL

Найдено вакансий: 100
Отфильтровано вакансий: 42

Топ-5 вакансий по зарплате:
Вакансия: Data Scientist (ML Engineer)
Компания: TechCorp
Зарплата: 250000-300000 RUR
Описание: Требуется опыт работы с Python, библиотеками для ML...
Ссылка: https://hh.ru/vacancy/123456
----------------------------------
```
